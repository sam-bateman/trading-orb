# Intraday Opening Range Breakout Strategy

A research project on the opening-range breakout (ORB) anomaly for US equities. Goal: build a systematic intraday strategy that holds up under honest out-of-sample testing.

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Alpaca API keys required for live data and paper trading.
